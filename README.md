# Portfolio Website

A responsive portfolio website built with HTML, SCSS, and JavaScript.

## Features

- Responsive design for all devices
- Project showcase section
- About and skills sections
- Contact form
- Smooth scrolling navigation

## Technologies

- HTML5
- SCSS
- JavaScript (ES6+)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/sharmanikhil88/nikhilsharma.github.io
.git
```

2. Install SCSS compiler:
```bash
npm install -g sass
```

3. Compile SCSS:
```bash
sass css/style.scss css/style.css --watch
```

4. Open `index.html` in your browser

## Project Structure
```
portfolio-website/
├── index.html
├── css/
│   ├── style.scss
│   └── style.css
├── js/
│   └── main.js
└── images/
```

## Customization

- Edit colors and fonts in `css/_variables.scss`
- Update content in `index.html`
- Modify styles in `css/style.scss`

## Deployment

Deploy to GitHub Pages, Netlify, or Vercel.
